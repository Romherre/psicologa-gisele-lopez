# Landing Page — Gisele López, Licenciada en Psicología

Sitio web freelance desarrollado para Gisele López, psicóloga clínica con enfoque psicoanalítico, especializada en adolescentes, adultos y parejas.

## 🔗 Demo en vivo
https://romherre.github.io/psicologa-gisele-lopez/

## 🧰 Stack
- HTML5 + CSS3
- Bootstrap 5
- JavaScript (validación de formulario)
- Formspree (envío de formulario de contacto)

## ✨ Secciones
- Banner principal con foto de fondo
- Sección "Sobre Gisele" con foto de perfil y experiencia profesional
- Formulario de contacto funcional
- Diseño responsive (mobile-first en la sección de contacto)
